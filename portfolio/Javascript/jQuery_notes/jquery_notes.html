jQuery Notes

Javascritpt behaves slightly differently in different browsers, so to smooth out some of the inconsistencies, jQuery, a Javascript framework, is used.

A framework is a file, or set of files, that you include in your project that contains existing code. Frameworks make it easier to build applications.

Methods

hide()

Description

Hides the matched elements

Parameters - hide(duration, complete)

duration

A string or number determining how long the animation will run

complete

A function to call once the animation is complete

Example Code

$("#input").next().hide();
